# Fund Returns Daily

Daily fund return dashboard built from KFR raw fund price data, `mapping.xlsx`,
and `Hana_EMP.xlsx`.

## Files

- `download_fund_price.py`: Logs in to KFR and downloads `펀드 기준가.xlsx`.
- `build_fund_dashboard.py`: Builds the standalone HTML dashboard.
- `docs/index.html`: GitHub Pages entry point generated by the build script.
- `mapping.xlsx`: Fund name/type/manager/launch-date mapping.
- `Hana_EMP.xlsx`: Manually maintained EMP cumulative return data.

## GitHub Actions setup

Create these repository secrets before running the scheduled workflow:

- `KFROM_ID`
- `KFROM_PASSWORD`

The workflow runs at 07:30 KST on Korean business mornings represented as
Sunday-Thursday 22:30 UTC. It downloads the latest raw file, rebuilds the HTML,
commits the updated files, and deploys `docs/index.html` to GitHub Pages.

Pushes that update `mapping.xlsx`, `Hana_EMP.xlsx`, or the dashboard scripts
rebuild and redeploy the page without downloading raw data.

In repository settings, set GitHub Pages to use **GitHub Actions** as the source.
